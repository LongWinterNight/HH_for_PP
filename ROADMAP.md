## 📊 Общая информация о проекте

**HH.ru Analytics** — профессиональная ETL-система для сбора, обработки и анализа вакансий с HH.ru с автоматическим извлечением навыков (Hard Skills, Soft Skills, Tools) и формированием аналитических отчётов.

### 1.3 Фильтры и Экспорт (В РАБОТЕ)

**Статус:** 🔄 В работе  
**Время реализации:** ~3 часа  
**Сложность:** Высокая

**Планируемый функционал:**

#### A. Фильтры аналитики

**Компоненты:**
1. **Период** (dropdown)
   - За сегодня
   - За неделю
   - За месяц
   - За квартал
   - За год
   - Все время
   - Custom (date picker)

2. **Домен** (dropdown с поиском)
   - Все домены
   - IT и разработка
   - Строительство
   - Продажи
   - Медицина
   - Образование
   - и т.д.

3. **Регион** (multi-select)
   - Москва
   - Санкт-Петербург
   - Другие регионы
   - Удалённая работа

4. **Опыт** (checkbox group)
   - Без опыта
   - 1-3 года
   - 3-6 лет
   - Более 6 лет

**Техническая реализация:**
```javascript
// filters data structure
analyticsFilters: {
    period: 'month',
    domain: '',
    regions: [],
    experience: [],
    salaryOnly: false
}
```

**API:**
```python
GET /api/analytics/kpi?period=month&domain=it&regions=moscow,spb
GET /api/analytics/top-skills?type=hard&domain=it&experience=3-6
```

---

#### B. Экспорт отчётов

**Форматы:**
1. **PDF** (для печати)
   - KPI метрики
   - Топ навыков
   - Графики (PNG)
   - Дата и время генерации

2. **Excel** (для анализа)
   - Лист 1: KPI
   - Лист 2: Hard Skills
   - Лист 3: Soft Skills
   - Лист 4: Tools
   - Лист 5: Сырые данные

3. **CSV** (для импорта)
   - vacancies.csv
   - skills.csv

**Кнопки:**
```html
<button @click="exportReport('pdf')">
    <i class="fas fa-file-pdf"></i> PDF
</button>
<button @click="exportReport('xlsx')">
    <i class="fas fa-file-excel"></i> Excel
</button>
<button @click="exportReport('csv')">
    <i class="fas fa-file-csv"></i> CSV
</button>
```

**API:**
```python
POST /api/export/analytics
Params: format (pdf|xlsx|csv), filters

GET /api/reports/list
GET /api/reports/download/{filename}
```

- ⏳ Экспорт генерируется < 5 сек
