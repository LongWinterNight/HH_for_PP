# 📖 HH.ru Analytics — Обзор проекта

## Что это?

**HH.ru Analytics** — профессиональная ETL-система для сбора, обработки и анализа вакансий с HH.ru с автоматическим извлечением навыков (Hard Skills, Soft Skills, Tools) и формированием аналитических отчётов.

## Ключевые возможности

### 📊 Аналитика в реальном времени
- **8 KPI метрик** с трендами (Средняя ЗП, Медиана, Вакансий, Навыков, Компаний, Регионов, Hard Skills, Soft Skills)
- **Топ-20 навыков** по категориям (Hard Skills, Soft Skills, Tools)
- **Фильтры аналитики**: период, домен, регион, опыт, зарплата
- **Sparklines** — мини-графики трендов в KPI-карточках (14 дней)

### 🔍 Поиск и фильтрация
- Автодополнение вакансий, регионов и навыков
- Расширенные фильтры (опыт, зарплата, навыки)
- Пагинация результатов
- Сортировка по зарплате и дате

### 📥 Экспорт данных
- **PDF** — отчёты с KPI и топ навыков
- **Excel** — 5 листов (KPI, Hard, Soft, Tools, Данные)
- **CSV** — сырые данные для импорта

### 🤖 Парсер вакансий
- Оптимизированный сбор данных с API HH.ru
- Кэширование запросов (SQLite, автоочистка >7 дней)
- Инкрементальное обновление (только новые вакансии)
- Прогресс в реальном времени
- **Защита**: обязательная настройка email перед запуском парсера

### 🎨 UX/UI
- **Hero-секция** с призывом к действию
- **Onboarding Wizard** — пошаговый гайд из 3 шагов
- **Skeleton-анимации** загрузки для всех секций дашборда
- **Vue.js 3** SPA (Single Page Application)
- **Chart.js** для графиков
- **Tailwind CSS** для стилей

## Технологический стек

| Компонент | Технология |
|-----------|-----------|
| **Backend** | FastAPI + Uvicorn (Python 3.10+) |
| **Frontend** | Vue.js 3 + Chart.js + Tailwind CSS (CDN) |
| **БД** | SQLite (dev) → PostgreSQL (prod) |
| **ORM** | SQLAlchemy 2.0 |
| **Парсер** | requests + tenacity (retry logic) |
| **Аналитика** | pandas + numpy + openpyxl |

## Архитектура (высокоуровневая)

```
┌─────────────────────────────────────────────────────┐
│  Пользователь (браузер)                              │
│  Vue.js 3 SPA → API запросы                         │
└──────────────────┬──────────────────────────────────┘
                   │ HTTP
                   ▼
┌─────────────────────────────────────────────────────┐
│  FastAPI (web/app/main.py)                           │
│  ├── /api/dashboard     → KPI, навыки, вакансии      │
│  ├── /api/parser/start  → запуск парсера (bg task)   │
│  ├── /api/analytics/*   → расширенная аналитика      │
│  ├── /api/export/*      → PDF/Excel/CSV              │
│  └── /api/user/email    → настройка email            │
└──────────┬──────────────────────────────┬────────────┘
           │                              │
           ▼                              ▼
┌──────────────────────┐    ┌──────────────────────────┐
│  SQLite (hh_vacancies│    │  APICache (api_cache.db) │
│  db)                 │    │  Кэш HH API запросов     │
│  ├── vacancies       │    │  (автоочистка >7 дней)   │
│  ├── parser_runs     │    └──────────────────────────┘
│  └── app_settings    │
└──────────────────────┘
           │
           ▼
┌─────────────────────────────────────────────────────┐
│  Оптимизированный парсер (optimized_parser.py)       │
│  ├── HH API client → сбор вакансий                  │
│  ├── Кэширование ответов                             │
│  └── Инкрементальный режим                           │
└─────────────────────────────────────────────────────┘
```

## Быстрый старт

```bash
cd hh_analytics
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python web/run.py
# Открой http://localhost:8000
```
