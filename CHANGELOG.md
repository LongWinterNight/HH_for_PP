# История изменений

Все заметные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
а проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.1.0] — 2026-03-29

### ✨ Добавлено

- **Оптимизированный парсер** (`optimized_parser.py`) с инкрементальным режимом
- **Кэширование API ответов** в SQLite для ускорения повторных запусков
- **Прогресс-бар** для визуализации процесса сбора
- **Векторизованный поиск навыков** с использованием regex
- **API endpoints** для управления кэшем (`/api/parser/cache/*`)
- Поддержка параметра `incremental` в веб-интерфейсе

### 🚀 Улучшено

- Ускорение парсинга в **6-10 раз** благодаря инкрементальному режиму
- Ускорение поиска навыков в **3-5 раз** благодаря векторизации
- Детальный парсинг только первых 2 страниц (вместо всех)
- Обновлён `README.md` с полной документацией

### 📦 Зависимости

- Добавлен `tqdm>=4.66.0` для прогресс-бара

### 📄 Документация

- Обновлён `README.md` с разделом об оптимизации
- Добавлен `CONTRIBUTING.md` для контрибьюторов
- Добавлен `CHANGELOG.md`
- Добавлен `LICENSE` (MIT)

## [1.0.0] — 2026-03-27

### ✨ Добавлено

- **ETL пайплайн** для сбора и обработки вакансий с HH.ru
  - Extract: Сбор через HH.ru API
  - Transform: Извлечение и классификация навыков
  - Load: Загрузка в SQLite
  - Analyze: Генерация отчётов

- **Веб-интерфейс** на FastAPI + Vue.js 3
  - Дашборд со статистикой
  - Поиск и фильтрация вакансий
  - Аналитика с группировками
  - Управление парсером
  - Экспорт отчётов

- **CLI интерфейс** (`db_cli`) для работы с базой данных

- **Расширенная аналитика** (`advanced_analyzer.py`)
  - Группировка технологий (20+ категорий)
  - Карта связей вакансия-навык
  - Детальные Excel-отчёты

- **Автоматическое извлечение навыков**
  - Hard Skills (518 навыков)
  - Soft Skills (74 навыка)
  - Tools (156 инструментов)

- **Словари навыков** в `config.yaml`
  - 58 поисковых запросов
  - Поддержка множественных языков

- **Тесты** (35 тестов)
  - Тесты API клиента
  - Тесты процессора
  - Тесты анализатора

### 📦 Зависимости

- `pandas>=2.0.0` — обработка данных
- `sqlalchemy>=2.0.0` — работа с БД
- `fastapi>=0.109.0` — веб-фреймворк
- `uvicorn>=0.27.0` — ASPI сервер
- `openpyxl>=3.1.0` — Excel отчёты
- `pymorphy3>=1.0.0` — лемматизация
- `pyyaml>=6.0` — конфигурация
- `pydantic>=2.0.0` — валидация
- `pytest>=8.0.0` — тестирование

### 📄 Документация

- `README.md` — основная документация
- `WEB_APP.md` — руководство по веб-приложению
- `requirements.txt` — зависимости
